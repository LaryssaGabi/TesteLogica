# Fibonacci Sequence 

Este projeto tem como objetivo calcular a sequência de Fibonacci e verificar se um número informado pertence a essa sequência.

## Como Funciona

A sequência de Fibonacci começa com 0 e 1, e cada número subsequente é a soma dos dois números anteriores. O programa permite que o usuário insira um número e verifica se ele está presente na sequência.

## Tecnologias Usadas

- Java

## Como Usar

1. Clone o repositório.
2. Abra o arquivo `Fibonacci` no seu IDE de preferência.
3. Compile o programa.
4. Execute o programa e insira um número quando solicitado.

# Soma de Números

Este projeto calcula a soma de números inteiros até um determinado índice utilizando um loop.

## Como Funciona

O programa inicializa com um índice e utiliza um loop para calcular a soma dos números inteiros de 1 até esse índice.

## Tecnologias Usadas

- Java

## Como Usar

1. Clone o repositório.
2. Abra o arquivo `Soma` no seu IDE de preferência.
3. Compile o programa.
4. Execute o programa e observe a soma calculada.
